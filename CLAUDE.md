# DRAUGHTS — контекст проекта для Claude

## Цель проекта

Переписать игру **русские шашки** с **Borland Pascal 7.0** (~1998–2000) на **Python 3.12+ / PyQt6**.
Полный технический разбор оригинала: **[ANALYSIS.md](ANALYSIS.md)**

---

## Команды разработки

```bash
# Запуск игры (без заставки — для быстрого тестирования)
python main.py --no-splash

# Запуск с заставкой
python main.py

# Скриншоты (сохраняются в .screenshots/)
python main.py --no-splash --screenshot window
python main.py --screenshot splash
python main.py --screenshot all

# Тесты (запускать перед коммитом)
python -m pytest tests/ -q

# Линтер + форматтер (ruff)
ruff check draughts/ tests/          # проверка
ruff check --fix draughts/ tests/    # автоисправление
ruff format draughts/ tests/         # форматирование

# Проверка типов (mypy)
mypy draughts/ --ignore-missing-imports
```

Конфигурация ruff и mypy — в `pyproject.toml`.

## Принятые решения

- **Фреймворк**: PyQt6 (единственная зависимость)
- **UI**: premium dark wood theme — процедурные текстуры (textures.py), 3D шашки, бархатное поле
- **Раскладка**: доска слева, нотация + кнопки справа (как в оригинале)
- **Окно**: изменяемый размер с пропорциональным масштабированием
- **Язык интерфейса**: только русский
- **Звук**: процедурно сгенерированные WAV, через QMediaPlayer
- **Форматы**: JSON (обратная совместимость: поле `movie` → `replay_positions`)
- **Автор**: Andrey Dugin (имя первым, не фамилия) — в заставке и «О программе»
- `IZVRAT.PAS` (10×10) — только после завершения 8×8 и по отдельному запросу

## Ключевые факты о коде

- Доска: `grid[y][x]`, символы `'b'/'B'/'w'/'W'/'n'`, индексация 1..8
- Тёмные клетки: `x % 2 != y % 2` → 32 активные клетки
- `y=1` — верх (чёрные), `y=8` — низ (белые); нотация `(1,1)=a8`
- Русские шашки: обязательное взятие, множественные прыжки, дамки на любое расстояние
- ИИ играет чёрными: `_see_beat()` → `_combination()` → `_action()` (приоритет)
- Подробности в [ANALYSIS.md](ANALYSIS.md)
