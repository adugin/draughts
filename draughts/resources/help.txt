РУССКИЕ ШАШКИ — СПРАВКА

ПРАВИЛА ИГРЫ

Русские шашки — настольная игра для двух игроков на доске 8x8.
Вы играете белыми (снизу), компьютер — чёрными (сверху).

Шашки ходят по диагонали на одну клетку вперёд.
Дамка (коронованная шашка) ходит на любое расстояние по диагонали.

Взятие обязательно. Если можно побить шашку противника — вы обязаны это
сделать. При взятии шашка перепрыгивает через вражескую по диагонали.
Возможны множественные взятия за один ход.

Шашка становится дамкой, достигнув последнего ряда доски.
Если превращение происходит во время серии взятий — шашка продолжает
бить уже как дамка.

Побеждает тот, кто побил все шашки противника или лишил его возможности
сделать ход.

Ничья объявляется при:
  - Троекратном повторении позиции.
  - 1 дамка против 1 дамки (недостаточно материала).
  - 2 дамки против 1 дамки (треугольник Петрова).
  - 15 ходов только дамками без взятий.
  - 30 ходов без взятий и движения шашек.

УПРАВЛЕНИЕ МЫШЬЮ

  Левая кнопка — выбрать шашку, указать позицию хода, промежуточное
                 взятие (всё через левый клик).
  Правая кнопка — отменить выбор шашки.

Чтобы сделать простой ход:
  1. Щёлкните левой кнопкой по своей шашке (она выделится).
  2. Щёлкните левой кнопкой по клетке, куда хотите пойти.

Чтобы сделать взятие:
  1. Щёлкните левой кнопкой по шашке, которая будет бить.
  2. Щёлкните левой кнопкой по конечной клетке — взятие автоматическое.
  3. Если путь неоднозначен — промежуточные точки указываются левым кликом.

Чтобы отменить выбор — щёлкните правой кнопкой или выберите другую шашку.

МЕНЮ

  Игра → Новая игра (Ctrl+N) — начать партию заново.
  Игра → Открыть (Ctrl+O) — загрузить сохранённую партию.
  Игра → Сохранить (Ctrl+S) — сохранить текущую партию в PDN.
  Игра → Отмена хода (Ctrl+Z) — отменить последний ход.
  Игра → Подсказка (Ctrl+H) — показать лучший ход.
  Игра → Сменить сторону (Ctrl+F) — поменяться цветом с компьютером;
         доска переворачивается, шашки игрока снизу.
         Доступно только на вашем ходе.
  Игра → Выход (Alt+F4) — выйти из программы.

  Настройки → Опции (Ctrl+,) — сложность, движок, интерфейс, анализ.
  Тренировка → Задачи (Ctrl+P) — решать тактические задачи.
  Вид → Просмотр партии (Ctrl+R) — воспроизвести все ходы.

  Позиция → Редактор (E) — расставить фигуры на доске.
  Позиция → Копировать FEN (Ctrl+Shift+C) — скопировать позицию.
  Позиция → Вставить FEN (Ctrl+Shift+V) — загрузить позицию.

  Анализ → Панель анализа (F3) — оценка текущей позиции.
  Анализ → Проанализировать партию — аннотации всех ходов.

  Справка → Информация (F1) — эта справка.
  Справка → Об авторе — информация об авторе программы.

НАСТРОЙКИ (ОПЦИИ)

Настройки сохраняются автоматически и загружаются при следующем запуске.

  Игра:
    Сторона       — играть белыми или чёрными.
    Уровень       — сила компьютера по шкале приблизительного Эло:
                    1=Новичок (~800), 2=Любитель (~1100), 3=Клубный (~1400),
                    4=Сильный клубный (~1700), 5=Кандидат (~2000),
                    6=Мастер (~2200+).
    Подсказка взятия — подсвечивать, когда взятие обязательно.

  Движок:
    Хэш-таблица    — размер таблицы транспозиций в МБ.
    Дебютная книга — использовать книгу дебютов (1572 позиции).
    Эндшпильная база — использовать базу эндшпилей (399K позиций).
    Глубина (dev)  — ручная глубина поиска (только для разработчиков).

  Интерфейс:
    Тема оформления — 7 тем на выбор.
    Скорость анимации — пауза между ходами для наглядности.
    Координаты — буквенно-цифровые метки клеток (a-h, 1-8).
    Подсветка хода — маркер последнего хода.
    Возможные ходы — точки допустимых ходов при наведении курсора.

  Анализ:
    Панель анализа — F3 или меню Анализ.
    Анализ партии — меню Анализ → Проанализировать партию.

РЕДАКТОР ПОЗИЦИЙ

  Вход: клавиша E или меню Позиция → Редактор.

  Левый клик — циклически: пусто → чёрная → чёрная дамка → белая →
               белая дамка → пусто.
  Правый клик — очистить клетку.

  Кнопки:
    ▶ Играть     — начать игру из текущей позиции.
    Анализ       — проанализировать текущую позицию.
    Отмена       — вернуться к предыдущей партии (или Escape).
    Очистить     — убрать все фигуры.
    Начальная    — стандартная расстановка.
    Импорт FEN   — загрузить позицию из текста.
    Экспорт FEN  — скопировать позицию в буфер обмена.

  Валидация при нажатии «Играть» или «Анализ»:
    - Пешки на последнем ряду автоматически превращаются в дамки.
    - Доска не может быть пустой.
    - У обеих сторон должны быть фигуры.

КОМАНДНАЯ СТРОКА

  python main.py                    — новая игра
  python main.py game.pdn           — загрузить партию из PDN
  python main.py --resume           — продолжить прерванную партию
  python main.py --difficulty 4     — установить уровень (1–6)
  python main.py --black            — играть чёрными
  python main.py --version          — показать версию

Параметры можно комбинировать:
  python main.py game.pdn --difficulty 3 --black

Приятной игры!
