РУССКИЕ ШАШКИ — СПРАВКА

ПРАВИЛА ИГРЫ

Русские шашки — настольная игра для двух игроков на доске 8x8.
Вы играете белыми (снизу), компьютер — чёрными (сверху).

Шашки ходят по диагонали на одну клетку вперёд.
Дамка (коронованная шашка) ходит на любое расстояние по диагонали.

Взятие обязательно. Если можно побить шашку противника — вы обязаны это
сделать. При взятии шашка перепрыгивает через вражескую по диагонали.
Возможны множественные взятия за один ход.

Шашка становится дамкой, достигнув последнего ряда доски.

Побеждает тот, кто побил все шашки противника или лишил его возможности
сделать ход.

УПРАВЛЕНИЕ МЫШЬЮ

  Левая кнопка — выбрать шашку или указать конечную позицию хода.
  Правая кнопка — отметить промежуточную позицию при множественном взятии.

Чтобы сделать простой ход:
  1. Щёлкните левой кнопкой по своей шашке (она выделится).
  2. Щёлкните левой кнопкой по клетке, куда хотите пойти.

Чтобы сделать взятие:
  1. Щёлкните левой кнопкой по шашке, которая будет бить.
  2. Если взятие одиночное — щёлкните левой кнопкой по конечной клетке.
  3. Если взятие множественное — отметьте промежуточные клетки правой
     кнопкой, а конечную — левой.

Чтобы отменить выбор шашки — щёлкните левой кнопкой по другой своей шашке.

МЕНЮ

  Игра → Новая игра (Ctrl+N) — начать партию заново.
  Игра → Открыть (Ctrl+O) — загрузить сохранённую партию.
  Игра → Сохранить (Ctrl+S) — сохранить текущую партию.
  Игра → Отмена хода (Ctrl+Z) — отменить последний ход.
  Игра → Выход (Alt+F4) — выйти из программы.

  Настройки → Опции (Ctrl+P) — сложность, глубина поиска, звук и другое.
  Вид → Просмотр партии (Ctrl+R) — воспроизвести все ходы текущей партии.
  Справка → Информация (F1) — эта справка.
  Справка → Об авторе — информация об авторе программы.

НАСТРОЙКИ (ОПЦИИ)

  Сложность — уровень игры компьютера (Любитель / Нормал / Профессионал).
  Подсказка взятия — подсвечивать, когда взятие обязательно.
  Звук — включить/выключить звуковые эффекты.
  Задержка — пауза между ходами для наглядности.
  Глубина поиска — глубина просчёта ходов (0 = авто, 1-10 = вручную).
  Играть чёрными — поменяться сторонами с компьютером.

Приятной игры!
