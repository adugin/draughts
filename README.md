# Русские шашки

Игра в русские шашки с компьютерным противником на базе alpha-beta minimax.

![Русские шашки — скриншот игры](docs/screenshot.png)

## Особенности

- **Классические русские шашки** — полная реализация правил FMJD: обязательное взятие, множественные прыжки, дамки на любое расстояние, правила ничьей (3-кратное повторение, 2K vs 1K, 15-ходовое правило для дамок, 30-ходовое правило)
- **6 уровней сложности** — от Новичка (~800 Elo) до Мастера (~2200+)
- **Алгоритмический ИИ** — alpha-beta pruning с TT (Zobrist), iterative deepening, quiescence search, LMR, killer moves
- **Дебютная книга** — 1572 позиции для быстрых и разнообразных начал
- **Эндшпильная база** — 399280 позиций (≤3 фигуры) для идеальной игры в окончаниях
- **Texel-тюнинг** — веса оценки оптимизированы автоматически через self-play
- **PDN 3.0** — сохранение и загрузка партий в международном формате
- **FEN** — копирование и вставка позиций через буфер обмена
- **Панель анализа** (F3) — оценка позиции, лучший ход, глубина, время
- **Анализ партии** — автоматические аннотации с графиком оценки
- **Тренажёр задач** — тактические задачи с автомайнингом из ваших партий
- **Редактор позиций** (E) — расстановка, FEN, валидация, «играть/анализировать отсюда»
- **7 тем оформления** — TOML-движок: dark wood, classic light, Nord, Dracula, Gruvbox, Catppuccin, Solarized
- **Реалистичная графика** — процедурные текстуры дерева, 3D-шашки с освещением
- **Сохранение настроек** — предпочтения сохраняются между сеансами

## Требования

- Python 3.12+
- PyQt6 >= 6.5
- NumPy >= 1.26

## Установка и запуск

```bash
# Клонирование
git clone https://github.com/adugin/draughts.git
cd draughts

# Установка зависимостей
pip install -r requirements.txt

# Запуск
python main.py
```

### Параметры командной строки

```bash
python main.py                    # Новая игра
python main.py game.pdn           # Загрузить партию из PDN-файла
python main.py --resume            # Продолжить прерванную партию
python main.py --difficulty 5      # Уровень сложности (1–6)
python main.py --black             # Играть чёрными
python main.py --version           # Показать версию
```

## Как играть

Вы играете **белыми** (снизу), компьютер — **чёрными** (сверху).

### Управление мышью

| Действие | Описание |
|----------|----------|
| **Левая кнопка** | Выбрать шашку / указать позицию хода / промежуточное взятие |
| **Правая кнопка** | Отменить выбор шашки |

### Простой ход

1. Щёлкните левой кнопкой по своей шашке — она выделится.
2. Щёлкните левой кнопкой по клетке, куда хотите пойти.

### Взятие (обязательно!)

1. Щёлкните левой кнопкой по шашке, которая будет бить.
2. Щёлкните левой кнопкой по конечной клетке — взятие выполнится автоматически.
3. Если путь неоднозначен — промежуточные точки указываются левым кликом.

### Правила

- Шашки ходят по диагонали на одну клетку вперёд
- Дамка ходит на любое расстояние по диагонали
- Взятие **обязательно** — если можно побить, вы обязаны это сделать
- Шашка становится дамкой, достигнув последнего ряда (продолжает бить как дамка)
- Побеждает тот, кто побил все шашки противника или лишил его возможности хода
- Ничья: 3-кратное повторение, 1K vs 1K, 2K vs 1K, 15 ходов только дамками, 30 ходов без взятий

|---------|----------|
| `Ctrl+N` | Новая игра |
| `Ctrl+O` | Открыть партию |
| `Ctrl+S` | Сохранить партию |

| `Ctrl+H` | Подсказка лучшего хода |

## Настройки

Доступны через меню **Настройки → Опции** (`Ctrl+,`). Сохраняются автоматически.

### Вкладка «Игра»
| Параметр | Описание |
|----------|----------|
| Сторона | Играть белыми или чёрными |
| Уровень | 1=Новичок (~800) ... 6=Мастер (~2200+) |
| Подсказка взятия | Подсвечивать обязательные взятия |

### Вкладка «Движок»
| Параметр | Описание |
|----------|----------|
| Хэш-таблица | Размер таблицы транспозиций (МБ) |
| Дебютная книга | Использовать книгу дебютов |
| Эндшпильная база | Использовать базу эндшпилей |

### Вкладка «Интерфейс»
| Параметр | Описание |
|----------|----------|
| Тема оформления | 7 тем на выбор |
| Координаты | Буквы и цифры вокруг доски |
| Подсветка хода | Маркер последнего хода |
| Возможные ходы | Точки при наведении курсора |
